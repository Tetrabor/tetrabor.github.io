What Is a Start Node?

The Start node marks the starting point of your workflow. The start node triggers from events such as end-users landing on the page, or a hand-off from another workflow. New workflows begin with a Start node in the Authenticated swimlane by default. Workflows can have multiple Start, Signal Start, or Timer Start nodes, but they require at least one of the three to function.

To learn more about the Start node, visit our Start Node article.
