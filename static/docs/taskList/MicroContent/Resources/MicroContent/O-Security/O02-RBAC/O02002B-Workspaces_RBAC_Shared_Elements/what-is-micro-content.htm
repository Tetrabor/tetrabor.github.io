What Are Shared Elements?

Shared elements include modules and reference data. Shared elements work with Workspaces Role-Based Access Control (WSRBAC) to limit which modules, data collections and workspaces creators can access. Other applications can also reference shared elements. For example, you can share an API module with other applications, rather than recreating the module for each.
