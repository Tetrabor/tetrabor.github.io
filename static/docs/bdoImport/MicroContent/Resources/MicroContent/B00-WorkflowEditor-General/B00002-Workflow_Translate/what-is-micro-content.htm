What Are Workflow Translate Settings?

As you build applications, you might need to deliver them in one or more languages. You can easily set up translations that display based on the end-user End-users, also known as Express Users, are the individuals accessing an application through Express View. In most cases, end-users are the customers using the product.'s location (determined by the data.currentUser.locale value). To translate module-level content, you'll use the Module Builder's Translate Settings. For workflows, you'll use the Workflow Builder Translate Settings. These settings let you translate the labels of Task nodes, navigation buttons, and so on.

To learn more about Workflow Translate Settings, visit our Workflow Translate Settings article.
