What Is a Multi-Select Dropdown Component?

The Multi-Select Dropdown component is a compact way to show a long list of options to your end-user End-users, also known as Express Users, are the individuals accessing an application through Express View. In most cases, end-users are the customers using the product.. The Multi-Select Dropdown component shares almost all functionality with the standard Dropdown component. But, the Multi-Select component lets your end-user make more than one selection.
