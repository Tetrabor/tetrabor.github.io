What Is the Release Management Dashboard Tool?

The Release Management Dashboard tool makes promoting releases to different Unqork environments easy. You can take your creations from one environment to the next, whether you're moving from testing or staging to QA or preprod to prod. The tool's features include:

  • Create releases with one or more applications including their modules and data collections.

  • Add detailed release information to make it easier to search for your releases. With the Release Management Dashboard tool, you can keep your promotions organized.

  • Roll back to a previous release if your application breaks. Then you can troubleshoot an issue without removing a working application from production.

To learn more about the Release Management Dashboard tool, visit our Release Management Dashboard Tool article.
